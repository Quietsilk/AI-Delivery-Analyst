### 1.3 Цветовой статус карточки по порогам

**Проблема:** при первом синке нет сигнала "хорошо / плохо", delta-стрелка появляется только со второго.

**Что сделать:**

Добавить CSS-класс `.kpi-card.kpi-status-good` и `.kpi-card.kpi-status-bad`:

```css
.kpi-card.kpi-status-good { background: rgba(62,207,142,0.05); }
.kpi-card.kpi-status-bad  { background: rgba(255,107,107,0.06); }
```

В `updateDashboard()` после заполнения значений добавить логику:

```js
// Cycle Time: хорошо < 5d, плохо > 10d
function setKpiStatus(cardId, value, goodThreshold, badThreshold) {
  const el = document.getElementById(cardId);
  el.classList.remove("kpi-status-good", "kpi-status-bad");
  if (value <= goodThreshold) el.classList.add("kpi-status-good");
  else if (value >= badThreshold) el.classList.add("kpi-status-bad");
}

setKpiStatus("kpi-cycle-time-card", d.cycleTimeDays, 5, 10);
setKpiStatus("kpi-lead-time-card",  d.leadTimeDays,  10, 20);
// Throughput: хорошо > 0
document.getElementById("kpi-throughput-card")
  .classList.toggle("kpi-status-bad", d.throughput === 0);
// Reopened: статус управляется существующим kpi-warn, не дублировать
```


**В CSS** добавить состояние свёрнутого сайдбара:
```css
.app-body.sidebar-collapsed { grid-template-columns: 0 minmax(0, 1fr); }
.app-body.sidebar-collapsed .sidebar { overflow: hidden; border: none; }
```

**В JS** добавить:
```js
const sidebarToggleBtn = document.getElementById("sidebar-toggle");
let sidebarCollapsed = LS.load("ada:sidebarCollapsed", false);

function applySidebarState() {
  document.querySelector(".app-body")
    .classList.toggle("sidebar-collapsed", sidebarCollapsed);
}

sidebarToggleBtn.addEventListener("click", () => {
  sidebarCollapsed = !sidebarCollapsed;
  LS.save("ada:sidebarCollapsed", sidebarCollapsed);
  applySidebarState();
});

// В restoreState() добавить:
applySidebarState();
```

**Автоколлапс после успешного Connect** — в `setConnected()`:
```js
function setConnected(ok, msg) {
  // ... существующий код ...
  if (ok) {
    sidebarCollapsed = true;
    LS.save("ada:sidebarCollapsed", true);
    applySidebarState();
  }
}
```

```html
  <div class="panel"> <!-- AI Analysis --> </div>
```
